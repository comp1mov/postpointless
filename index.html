post.pointless help
post.pointless is a browser-based point cloud viewer and studio sketch. Load a scan, navigate inside it, shape the cloud with points, noise, slice, lines and camera motion, then export images or video.
Capture
UI dot
Toggle UI. If Rec with UI Button is on, hold to record and release to stop.
R
Start or stop video recording.
J
Export JPEG using the current output setting.
View
H
Toggle interface visibility.
G
Toggle grid.
L
Lock or unlock camera controls.
0
Reset camera and cloud rotation.
1 / 2 / 3
Front, top, and side views. Current view is saved first.
4
Return to the saved camera position.
Navigation
W A S D
Move camera and target.
Q / E
Orbit around the target.
Space
Move up.
Shift
Fast movement or down movement depending on current navigation state.
